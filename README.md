## Development

### Backend
* `cd backend`
* run `npm install`
* run `npm run watch` to start server in development mode
* `npm test` to run unit tests
* `npm run build` to build server to dist directory

### Web Interface
* `cd web`
* run `npm install`
* run `npm start` to start dev webserver
* build to */web/dist* with `npm run build`
